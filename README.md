# How to use GorillaAlbums

GorillaAlbums may be difficult for users to use so here is a guide!

1. Launch your game with GorillaAlbums.dll

2. Go to Gorilla Tag > Mods > AlbumCovers (an intial launch action should be taken to ensure the folders initiate)
*this mod uses MelonLodaer **NOT** Bepinex

3. Inside that folder should be 4 subfolders, Album1, Album2, Album3, Album4. 
 
4. In each folder, you must place 1 image, if you don't the mod will display an error message in game.
*if any errors occur with this, please report a issue [here](https://github.com/elligurt/GorillaAlbums-ML/issues)*

This mod is currently in development, please join this [discord](A2BFDNK9yw) if you want to help give suggestions or need support using the mod

# Why do my albums display "GorillaAlbums" with nothing on it?

This means you don't have any pictures, not enough, or the wrong image type please follow the instructions above

**Supported Image Types**
- PNG
- JPG
*more may be supported in the future*
